# Bermuda Bluewater Guides Website

A ready-to-upload website for a Bermuda fishing guide / charter service.

## Easiest launch option: Vercel

1. Create a free GitHub account.
2. Create a new repository called `bermuda-fishing-site`.
3. Upload all files from this folder into that repository.
4. Create a free Vercel account.
5. Click **Add New Project** and import the GitHub repository.
6. Click **Deploy**.

## Change the business info

Open `src/App.jsx` and replace:

- `Bermuda Bluewater Guides`
- `bookings@bermudabluewaterguides.com`
- `+1 441 555 0192`
- Prices and trip names
- Testimonials

## Run locally, optional

```bash
npm install
npm run dev
```
